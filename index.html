SerenMind (清心) 隐私政策

生效日期：2026年7月5日

1. 信息收集

SerenMind (清心) 是一款心理健康自助工具。我们遵循"数据最小化"原则，不收集任何可识别您个人身份的信息（如姓名、手机号、邮箱地址）。

我们仅在必要时收集以下匿名数据：

• CBT 分析记录：您输入的思维文本和分析结果，仅存储在您的设备本地。

• 情绪记录：您选择的心情等级（1-5分），仅存储在您的设备本地。

• 呼吸练习记录：练习时长和类型的统计数据，仅存储在您的设备本地。

• 语音输入：仅在您主动使用语音输入功能时，经您授权后访问麦克风。录音数据仅用于语音转文字，不会上传或存储到任何服务器。您可以在系统设置中随时关闭麦克风权限。

• 设备网络信息（SSID/BSSID）：通过 Firebase Analytics 自动获取您当前连接的 WiFi 网络名称（SSID）和基站标识（BSSID），仅用于估算设备所在的国家/城市级别粗略地理位置，以统计不同地区用户的使用分布。此信息为 Firebase SDK 默认采集的匿名统计数据，不与任何个人身份关联，也不会被存储或分享给其他第三方。您可以在首次启动时拒绝同意隐私政策，或通过卸载 App 撤回同意。

• 匿名行为数据：通过 Firebase Analytics 收集非个人身份的使用统计数据（如功能使用频率、设备型号、操作系统版本）。以上均为匿名统计数据，不包含个人身份信息。

2. AI 分析说明

当您使用 CBT 分析功能时，您输入的思维文本会被发送到 DeepSeek API 服务器进行处理。此过程中：

• 您的文本不与任何个人身份信息关联。

• DeepSeek 不会将您的文本用于模型训练（根据其 API 条款）。

• 分析结果会自动保存到您的设备本地，不会上传到我们的任何服务器。

3. 数据存储

所有 CBT 记录、情绪数据、呼吸记录均仅存储在您设备的 Room 本地数据库中。我们不运营任何后端服务器来收集或存储您的个人数据。

您可以随时在 App 中删除这些数据。卸载 App 后，所有本地数据将被永久移除。

4. 第三方服务

SerenMind (清心) 使用以下第三方服务：

• Firebase Analytics（Google）：收集匿名统计数据，具体包括：功能使用频率、设备型号、操作系统版本、应用使用时长，以及 WiFi 网络名称（SSID）和基站标识（BSSID）用于粗略地理位置估算（仅到城市级别）。以上数据均为匿名统计，不包含个人身份信息。您可以随时通过卸载 App 撤回同意。

• Firebase Cloud Messaging（Google）：用于发送可选的推送通知（如每日提醒）。您可以在 App 中随时关闭推送。

• DeepSeek API：用于 CBT 分析功能。您的思维文本在分析时发送至 DeepSeek 服务器，不与任何个人身份信息关联。

5. 用户权利

您拥有以下权利：

• 访问：在"历史记录"页面查看所有本地存储的数据。

• 删除：随时在 App 中逐条删除或一键清空所有数据。

• 导出：通过"导出报告"功能将 CBT 分析记录导出为 PDF。

• 撤回同意：卸载 App 即撤回所有同意。

6. 儿童隐私

SerenMind 不面向 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。
