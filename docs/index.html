清心 (QingXin) 隐私政策

生效日期：2026年6月14日

1. 信息收集

清心是一款心理健康自助工具。我们遵循"数据最小化"原则，不收集任何可识别您个人身份的信息（如姓名、手机号、邮箱地址）。

我们仅在必要时收集以下匿名数据：

  • CBT 分析记录：您输入的思维文本和分析结果，仅存储在您的设备本地。
  • 情绪记录：您选择的心情等级，仅存储在您的设备本地。
  • 呼吸练习记录：练习时长和类型的统计数据，仅存储在您的设备本地。
  • 匿名行为数据：通过 Firebase Analytics 收集非个人身份的使用统计数据。

2. AI 分析说明

当您使用 CBT 分析功能时，您输入的思维文本会被发送到 DeepSeek API 服务器进行处理。此过程中：

  • 您的文本不与任何个人身份信息关联。
  • DeepSeek 不会将您的文本用于模型训练。
  • 分析结果会自动保存到您的设备本地。

3. 数据存储

所有记录均仅存储在您设备的 Room 本地数据库中。我们不运营任何后端服务器来收集或存储您的个人数据。卸载 App 后，所有本地数据将被永久移除。

4. 第三方服务

  • Firebase Analytics：收集匿名行为统计数据。
  • Firebase Cloud Messaging：用于可选的推送通知。
  • DeepSeek API：用于 CBT 分析功能。

5. 用户权利

  • 访问：在 App 中查看所有本地存储的数据。
  • 删除：随时在 App 中删除数据。
  • 导出：导出 CBT 分析记录为 PDF。
  • 撤回同意：卸载 App 即撤回所有同意。

6. 儿童隐私

清心不面向 13 岁以下儿童。
